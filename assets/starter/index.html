DSGo Starter

A guided tour of every v1 bridge surface with live demos and copy-pastable code on each page.

What you’re looking at

This app is running inside a sandboxed iframe-free runtime at /apps/dsgo-starter/*. Each route below is a real, indexable HTML page served by the DSGo Apps plugin.

The plugin injects a global window.dsgo client. Every method is permission-checked server-side against the manifest. Open any page below to see the call, the response, and the producing code.
