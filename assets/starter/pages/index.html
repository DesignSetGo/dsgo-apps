Pages

Read published WordPress pages. Same returned shape as posts — categories and tags are present but typically empty for pages. Use the orderby/search options to navigate the hierarchy your theme exposes.

        Same shape as posts
        Page records carry the same fields as posts (id, slug, title, excerpt, content, status, protected, date, modified, author, link, featured_media_url, categories, tags). For pages, categories/tags are usually empty arrays.
